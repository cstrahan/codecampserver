Prince is a computer program that converts XML and HTML into PDF documents by way of CSS. Prince runs on major operating platforms such as Windows, Linux, and MacOS X. For extensive information about Prince, please visit www.princexml.com

Using Prince

Run Prince from the Start Menu item "Prince", or by double-clicking on the Prince icon on your desktop.

The main window has five tab pages, making it easy for you to navigate through different areas of the application.

1. Documents Page

You can add documents here for conversion. Although the default document types are .xml, .xhtml, .html or .htm, Prince does not really care about the file extension as long as it is a valid XML document. To get started, you can find sample documents on our website.

Once a document has been added, you start the conversion process by clicking on the Go button. Once a document is processed and a PDF file is produced successfully, its status will change to "Converted". The output PDF file is saved in the same directory as the input XML document.

If the status of the currently selected document is "Converted", an icon representing the PDF file produced from that document will be displayed on the right hand side of the page. You can click on the View button to view it.

The status of a document can be one of the following three:

  • Unconverted: no attempts have be made to convert it
  • Converted: document has been converted successfully
  • Unsuccessful: conversion has been unsuccessful and no PDF file was produced

You can view the output log on the Output Log page for details of the conversion.

2. Style sheets Page

This page is used to apply CSS style sheets to documents. If your document references style sheets, these style sheets will be applied automatically and there is no need to use this page.

3. Configuration Page

On this page you can select a document type for the document to be converted and enter encryption information if you want to encrypt the PDF file for security reasons. Encryption key can be 40 bits or 128 bits. Note that not all PDF viewers support encryption.

By default the document type will be auto-detected, which should work fine for most XML and HTML documents.

4. Output Log Page

The Output Log page displays the messages Prince produced during the conversion. Messages are categorized into three major groups:

  • Information: general information the user might be interested in
  • Warning: minor problems encountered during conversion
  • Error: problems that have had a major impact on the conversion

5. Preview Page

When a PDF file is successfully produced, it can be viewed on this page. You will need to have Acrobat Reader installed on your computer to be able to use this feature. You can view one PDF file at a time. The previous PDF file will be closed when you open and view a new one. The View button appears on the lower right hand side of the Documents page when you click on a document that has a status of "Converted".

Licenses

To purchase a license to use Prince for commercial purposes, please visit our website.

After purchasing a license, a license file will be emailed to you. Save this file in any directory of your choice. The License window can be brought up from the License menu item in the Help menu. You can simply drag and drop the license file to this window from whatever location you might have saved it (or you can use the Open button to open it if you are not used to drag-and-drop). Read the license details. If you agree with the license, press the Accept button to activate the new license for Prince.
